# ランク別制限緩和表

本表は、生活ワールドにおけるサーバー負荷を適切に管理し、全てのプレイヤーが快適に動作する環境を維持するための基準です。

**制定日：空白**
**最終改定日：2026年3月7日**

---

## 生活ワールド エンティティ制限基準

| ランク | 1チャンク内 エンティティ上限 |
|--------|------------------------------|
| なし（見習い） | **15** 個 |
| 新米建築家 | **25** 個 |
| 中堅建築家 | **40** 個 |
| 熟練建築家 | **60** 個 |
| 筆頭建築家 | **90** 個 |
| 匠（たくみ） | **120** 個 |

---

## 制限緩和に関する補足事項

### エンティティの定義と管理
額縁、防具立て、絵画、吊り看板、家畜、村人、ボート、トロッコ等を指します。

> **特に「村人」や「家畜」の密集はサーバーへの負荷が非常に高いため、ランクに関わらず1チャンク内に30体以上の生物を配置する場合は、運営へ事前に相談してください。**

### 制限超過時の対応
定期的な負荷チェックにより、ランク上限を著しく超える設置が確認された場合、付近のTPS（処理速度）低下の原因となっている箇所について、運営により整理・撤去の要請を行う場合があります。

### ショップ区画との違い
本表は生活ワールド専用です。ショップ区画においては、本表よりも緩和された「ショップ運営ガイドライン」の制限値が優先的に適用されます。
